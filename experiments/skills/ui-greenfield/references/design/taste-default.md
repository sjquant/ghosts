# Taste Default

Use for operational UI: internal tools, dashboards, admin flows, data-heavy pages, and "make it usable" briefs.

## Goals

- clarity
- density without clutter
- predictable hierarchy
- accessible states
- fast comprehension
- low surprise

## Visual rules

- Prefer restrained palettes.
- Use one primary accent.
- Use clear spacing rhythm.
- Keep typography functional.
- Make tables, filters, forms, and empty states excellent.
- Do not add decorative complexity that hurts task flow.

## Do not use when

The brief signals premium, glossy, brand-grade, beautiful, wow, or a named high-craft product company.
