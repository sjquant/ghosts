# Brand Reference - Vercel

## Atmosphere

Sharp, monochrome, platform-native, typographically disciplined, high-confidence.

## Color

- black/white/neutrals
- subtle gray scale
- accent only when meaningful
- high contrast

## Typography

- precise sans-serif
- strong scale
- tight alignment
- code-friendly mono when needed

## Components

- sharp cards
- clean CTAs
- deployment/status surfaces
- documentation blocks
- platform navigation

## Material

Depth comes from contrast, spacing, borders, and typographic precision rather than heavy decoration.

## Agent guide

Use for developer platforms, deployment tools, documentation-driven products, and monochrome premium UI. Do not copy Vercel logos or copy.
