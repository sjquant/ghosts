# Brand Reference - Linear

## Atmosphere

Precise, calm, product-focused, engineered, quietly premium.

## Color

- near-black or deep neutral backgrounds for dark mode
- off-white foreground
- restrained blue/purple gradient accents
- subtle borders and low-opacity separators
- avoid loud saturation

## Typography

- modern sans-serif
- tight but readable hierarchy
- medium weights for UI
- careful letter spacing in large headings

## Components

- compact buttons
- clean command surfaces
- fine borders
- subtle cards
- refined sidebars and lists

## Material

Minimal depth, but not flat: use fine borders, slight gradients, subtle shadows, and accent glow sparingly.

## Agent guide

Use for Linear-like, precise, minimal productivity software. Do not copy Linear logos, text, or exact assets.
