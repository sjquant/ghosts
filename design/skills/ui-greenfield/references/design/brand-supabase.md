# Brand Reference - Supabase

## Atmosphere

Developer-native, dark, code-first, practical but premium. Database/platform energy with emerald accents.

## Color

- dark neutral canvas
- emerald green accent
- code surface neutrals
- soft glow for important product moments

## Typography

- clean sans for UI
- monospace for code/data surfaces
- clear technical hierarchy

## Components

- code blocks
- dashboard cards
- API snippets
- status indicators
- product panels

## Material

Dark layers, terminal/code surfaces, fine borders, soft emerald glow, restrained depth.

## Agent guide

Use for developer tools, databases, infrastructure, APIs, and technical dashboards. Do not copy Supabase logos or copy.
