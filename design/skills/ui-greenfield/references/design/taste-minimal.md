# Taste Minimal

Use for minimal, editorial, quiet, precise, restrained, Linear-like, or Notion-like products.

## Goals

- disciplined spacing
- typographic hierarchy
- quiet confidence
- clear interaction states
- refined microcopy
- fewer effects, better alignment

## Rules

- Use contrast, spacing, and typography as the main materials.
- Keep surfaces quiet but not dead.
- Hover/focus states still need visible treatment.
- Use subtle gradients or borders only when they clarify structure.

## Failure conditions

- anonymous monochrome page with no hierarchy
- too much whitespace with no rhythm
- missing states
- vague gray-on-gray contrast
- no clear primary action
