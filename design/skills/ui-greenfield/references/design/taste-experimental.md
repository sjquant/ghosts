# Taste Experimental

Use for cinematic, bold, immersive, art-directed, scroll-driven, or Awwwards-like briefs.

## Goals

- memorable composition
- strong motion concept
- high-contrast visual idea
- deliberate surprise
- brand-safe experimentation

## Rules

- Define the central visual metaphor before code.
- Use motion to explain or delight, not distract.
- Keep accessibility and reduced motion.
- Progressive enhancement is preferred.
- Browser QA is mandatory.

## Failure conditions

- animation with no concept
- unreadable text
- performance collapse
- inaccessible contrast
- effect-heavy but brand-empty output
