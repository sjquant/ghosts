# AI 한글 티 분류 체계

LLM이 생성한 한국어 글에서 반복되는 AI 티를 10개 대분류로 정리한다. 각 항목은 탐지 기준, 심각도, 윤문 방향만 남긴다.

## 심각도

- `S1`: 결정적 신호. 발견하면 제거한다.
- `S2`: 반복되면 강한 신호. 밀도를 낮춘다.
- `S3`: 단독으로는 약하지만 다른 패턴과 겹치면 조정한다.

## A. 번역투

| ID | 심각도 | 탐지 | 처방 |
|---|---|---|---|
| A-1 | S1 | `~에 대해(서)` | 목적격 조사로 직결한다. |
| A-2 | S1 | `~를 통해/통하여` 남발 | `~로`, `~해서`, `~함으로써`로 분산한다. |
| A-3 | S1 | `~에 있어(서)` | `~에서`, `~을 볼 때`로 바꾼다. |
| A-4 | S2 | `~라는 점에서` 반복 | `~서`, `~라는 이유로`로 줄인다. |
| A-5 | S2 | `~와 관련하여/관련된` | `~에`, `~의`, 목적격 조사로 줄인다. |
| A-6 | S2 | `~에 기반하여/바탕으로` 반복 | `~로`, `~을 보고`, `~를 근거로`로 바꾼다. |
| A-7 | S1 | `가지고 있다`, `회의를 가지다`, have/make/take/give 직역 | 동사·형용사로 환원한다. |
| A-8 | S1 | 이중 피동 `~되어진다`, `~지게 된다` | 능동 또는 단일 피동으로 바꾼다. |
| A-9 | S2 | `~에 의해` 피동 | 행위자를 주어로 올린다. |
| A-10 | S2 | `~할 수 있다` 남발 | 확정 가능한 문장은 단언한다. |
| A-11 | S2 | `~을 위해` 목적절 남발 | `~려고`, `~위한`, 직접 절로 바꾼다. |
| A-12 | S2 | `만들어지다`, `이루어지다` 자동 피동 | 행위자나 동사를 복원한다. |
| A-13 | S2 | 조사 없는 영어식 명사 나열 | 필요한 조사를 복원하거나 절로 푼다. |
| A-14 | S2 | 문두 `그리고`로 평문 연결 | `-고`, `-며`, `-면서`로 압축한다. |
| A-15 | S2 | 추상 주어 + `보여준다/제공한다/가져온다/시사한다` | 구체 주어, 원인절, 근거절로 바꾼다. |
| A-16 | S1 | 단락 안 `그/그녀/그것/그들` 반복 | 생략하거나 명사·호칭으로 바꾼다. |
| A-18 | S2 | 명사 앞 긴 관형구·관계절 좌향 수식 | 문장을 나누거나 후치 동격절로 푼다. |
| A-19 | S2 | `~에서의/~에로의/~으로의/~에의/~으로부터의` | 절로 풀어 쓴다. 단순 `~의`는 제외한다. |

## D. AI 관용구

| ID | 심각도 | 탐지 | 처방 |
|---|---|---|---|
| D-1 | S1 | `결론적으로`, `따라서`, `이를 통해`, `그러므로`, `요약하면`, `정리하면` 반복 | 대부분 삭제하고 필요한 1-2개만 변주한다. |
| D-2 | S1 | `시사하는 바가 크다`, `주목할 만하다`, `간과할 수 없다` | 삭제하거나 구체 결론으로 바꾼다. |
| D-3 | S1 | `본질적으로`, `핵심적으로` | 대비에 필요하지 않으면 삭제한다. |
| D-4 | S1 | `혁신적`, `획기적`, `압도적`, `강력한` 같은 hype 어휘 반복 | 삭제하거나 원문 사실로 대체한다. |
| D-5 | S1 | `기술이 묻는다`, `시대가 부른다` 같은 추상 주어 의인화 | 사람, 기관, 평범한 주어로 되돌린다. |
| D-6 | S1 | `~할 때다`, `~해야 한다`, `지금이야말로` 공식 결말 | 가능하면 평서형으로 닫는다. |
| D-7 | S2 | `X에서 Y로`, `X을 넘어 Y로` 반복 | 하나만 남기고 일반 서술로 바꾼다. |

## E. 리듬과 종결

| ID | 심각도 | 탐지 | 처방 |
|---|---|---|---|
| E-1 | S2 | 문장 길이가 지나치게 균일함 | 짧은 문장과 긴 문장을 섞는다. |
| E-2 | S2 | `~다` 종결 4회 이상, `~고 있다` 남발 | 종결과 시제를 섞고 단순 시제로 환원한다. |
| E-3 | S2 | 모든 문단이 3-4문장으로 균일함 | 1문장 문단과 긴 문단을 섞는다. |

| E-5 | S2 | 쉼표로 긴 절을 계속 잇는 문장 | 마침표로 나누거나 한국어식 절로 압축한다. |
| E-6 | S2 | 쉼표 위치가 무차별적으로 다양함 | 주절 경계나 명백한 삽입에만 둔다. |
| E-7 | S2 | 대화·구어의 말높임 불일치 | 한 화자·문단 안 말높임을 일관시킨다. |
