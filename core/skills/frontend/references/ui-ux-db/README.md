# UI/UX DB

Local lookup database for palettes, font pairings, and UX heuristics.

## Search

```bash
node scripts/search-uiux-db.mjs "premium fintech"
node scripts/search-uiux-db.mjs "dashboard table"
node scripts/search-uiux-db.mjs "wellness palette"
```

## Rule

This is a lookup tool, not the design source of truth. Use results to update the selected design source or implementation notes; do not bypass the design reference workflow.
