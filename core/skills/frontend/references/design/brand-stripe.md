# Brand Reference - Stripe

## Atmosphere

Polished, trustworthy, premium SaaS. Colorful but disciplined, with strong hierarchy and dimensional marketing sections.

## Color

- multi-stop gradients
- blue/purple/indigo accent families
- crisp foreground/background contrast
- semantic surfaces for trust and clarity

## Typography

- strong display hierarchy
- readable body text
- confident CTA labels

## Components

- dimensional hero panels
- polished CTAs
- layered product mockups
- trust badges
- content cards with clear depth

## Material

Use gradients, soft shadows, layered panels, and luminous accents. Avoid single flat fills.

## Agent guide

Use for premium SaaS, fintech, checkout, billing, API platforms, and high-trust landing pages. Do not copy Stripe logos, trademarks, or copy.
